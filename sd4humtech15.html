Two ways of looking at the humanitarian world

Closed World Assumption

  • If something is not known about a domain, it is assumed to be false
  • Relational databases reinforce these assumptions
  • Database schema change required to incorporate new knowledge

Open World Assumption

  • What is not known is simply still unknown
  • New knowledge can be seamlessly incorporated as it is learned
  • Extend the core vocabularies as needed

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name.
